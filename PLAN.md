# خطة مصنّف الذاكرة — الإطلاق والتطوير

## المرحلة 1: الإطلاق المجاني (أول شهرين)
- PWA مجاني بالكامل — 200 صورة/يوم لكل مستخدم
- تسويق في جروبات فيسبوك (قطع غيار، مهندسين صيانة)
- فيديو 30 ثانية للتسويق
- الهدف: 200-500 مستخدم حقيقي
- التكلفة: ~$110-150/شهر (Vision API)

## المرحلة 2: Google Play + اشتراك (شهر 3-4)
- TWA لتحويل PWA لـ APK
- 30 صورة/يوم مجانية، فوقها اشتراك 30-50 جنيه/شهر
- دفع: فودافون كاش / إنستاباي

## المرحلة 3: تأمين الكود
- نقل DB للسيرفر فقط (أهم حاجة)
- JavaScript Obfuscation
- API protection (كل حاجة تعدي على السيرفر)

## المرحلة 4: تطوير الأدمن + UX (شهر 2-3)
- Dashboard بأرقام حية
- إشعارات Telegram للصور الفاشلة
- تصنيف الصور الفاشلة من الأدمن + إضافة للـ DB
- أصوات مختلفة لكل نوع
- مشاركة النتيجة
- وضع batch

## المرحلة 5: التوسع (شهر 5+)
- شراكات مع محلات كبيرة
- API مدفوع
- أنواع شرائح جديدة (UFS, DDR5)
- خطة Enterprise
- Offline mode محسّن
